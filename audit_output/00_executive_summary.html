Executive Summary

Date: 2026-03-28  •  Scope: Full codebase audit — agent logic, engagement flow, UI/UX

The Single Biggest Logic Risk

FIXED — Synchronous Claude API calls block the entire application. Timeouts have been added to all Anthropic API calls: coach.py (30s), psych_intake.py (30s chat / 60s report), app.py photo analysis (45s). The app no longer hangs indefinitely on slow Claude responses.

The Single Biggest Engagement Opportunity

FIXED — The Coach is hidden behind a floating button. The Coach's latest message is now displayed inline on the main screen with quick-reply buttons ("How should I modify today?" / "I'm struggling" / "I crushed it"). The Coach is now the centerpiece of the daily experience, not hidden behind a FAB.

Three Highest-ROI UI/UX Changes

1. Auto-select today + move workout above metrics bars DONE

  • Effort: 2 hours
  • Impact: The workout is buried below 7 information bars. Main screen reordered: workout/readiness first, coach inline below, weigh-in/garmin/supplements at bottom. Today auto-selects in the day grid.

2. Add "Skip" and "Continue Later" to blocking overlays DONE

  • Effort: 4 hours
  • Impact: The morning check-in and psych intake both BLOCK the entire app with no escape. Morning check-in now has a Skip button. Psych intake has Continue Later button, progress indicator ("Getting to know you (X of ~12)"), and "Welcome back" on resume. INTAKE_LOCKED shows countdown with close button.

3. Inline Coach message with quick-reply prompts DONE

  • Effort: 1 day
  • Impact: Coach's latest message now shows inline on the main screen with quick-reply buttons. The Coach is the product's core interaction loop.
