What's Good

  1. Dark theme with high contrast — readable, professional, matches a gym/performance aesthetic
  2. Color-coded run types — Z2 blue, tempo amber, HIIT red, long purple — excellent at-a-glance differentiation
  3. Garmin metrics grid — compact, informative, color-coded by readiness score
  4. Morning check-in sliders — fast to use, conditional follow-ups are smart
  5. Meal plan with macro tracking — detailed but scannable, portion adjustment buttons are practical
  6. Phase/week/day navigation — logical hierarchy

What's Broken

  1. Information overload on the main screen FIXED — Main screen reordered: workout/readiness first, weigh-in/garmin/supplements at bottom
  2. No visual hierarchy for "what should I do RIGHT NOW" FIXED — Workout and readiness are now the hero elements
  3. Footer is wasted space FIXED — Footer removed, migration banner removed
  4. Chat FAB is the only way to reach the Coach FIXED — Inline coach message with quick-reply buttons on main screen
  5. Progress dashboard is orphaned FIXED — Progress dashboard button added to header
  6. No visual distinction between today and other days FIXED — Today auto-selected in day grid
  7. Psych intake chat is crammed into a card FIXED — Full-screen psych intake with full viewport flexbox layout
  8. Supplement tracker is flat and forgettable — Small toggle pills that blend into the surface.
  9. No empty states — Bars render with "--" or "?" with no guidance on what to do.
  10. Destructive settings with no confirmation FIXED — confirm() added to Redo Baseline and Redo Psych Intake

Component-Level Recommendations

Remove

  1. Static footer — DONE — removed
  2. Migration banner — DONE — removed
  3. Separate check-in summary bar — merge into the Coach's morning briefing
  4. Phase nav AND week tabs AND day grid as separate elements — merge into a unified navigation

Add

  1. "Today" hero card — auto-selects today, shows workout + readiness at a glance
  2. Inline Coach message — latest Coach message visible on main screen without opening overlay
  3. Progress sparklines — mini charts in the weigh-in bar and lift weight inputs
  4. Milestone banners — brief celebrations for PRs, streaks, weight milestones
  5. Coach quick-reply buttons — pre-written prompts below the inline Coach message
  6. Skip/dismiss on morning check-in — add "Skip today" that records a blank check-in
  7. Empty state guidance — "Connect Garmin to unlock readiness tracking" with benefit explanation
